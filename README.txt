# PageRank Algorithm - Web Crawler & Ranking

This project implements the PageRank algorithm to rank web pages based on links between them. The program crawls the web, retrieves HTML content, and calculates PageRank for each page.

## Introduction

This project simulates the PageRank algorithm, used by search engines to rank web pages. The algorithm assigns a rank to each page based on the number and quality of links to the page.

## Requirements

- Python 3.x
- SQLite (for storing web page data)
